#+TITLE: Hyperfocus-Isolated Org
#+CATEGORY: Concept Info Organization Types > Hyperfocus Isolated

* Definition
Hyperfocus-isolated organization hides all layers not currently in focus, presenting only the target concept layer to eliminate visual and cognitive distraction. It dynamically collapses unrelated layers based on the user's current focus target.

* ADHD/Hyperfocus/Russian Doll Compatibility
This structure is optimized for ADHD hyperfocus, using Russian Doll nesting to hide all layers except the target. By isolating a single layer, it removes all external cues that could trigger distraction, allowing hyperfocus to fully engage. The isolation is dynamic, shifting as hyperfocus moves between layers, matching the fluid nature of ADHD attention.
