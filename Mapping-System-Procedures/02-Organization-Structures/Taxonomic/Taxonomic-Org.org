#+TITLE: Taxonomic Org
#+CATEGORY: Concept Info Organization Types > Taxonomic

* Definition
Taxonomic organization classifies information into nested, hierarchical categories following formal classification rules (similar to biological taxonomy). It uses mutually exclusive categories with clear inclusion criteria at each level.

* ADHD/Hyperfocus/Russian Doll Compatibility
Taxonomic structures align with Russian Doll nesting through strict hierarchical categorization, where each layer is a subcategory of the layer above. For ADHD hyperfocus, taxonomic clarity eliminates ambiguity about where information belongs, reducing decision fatigue. Hyperfocus locks onto a single taxonomic branch, with other branches hidden to prevent distraction.
