#+TITLE: Hierarchical Nested Org
#+CATEGORY: Concept Info Organization Types > Hierarchical Nested

* Definition
Hierarchical nested organization structures information in nested layers similar to Russian dolls, where each outer layer encapsulates inner layers of increasing granularity. It follows parent-child relationships, with each layer inheriting context from the layer above.

* ADHD/Hyperfocus/Russian Doll Compatibility
This structure is the core Russian Doll implementation, with each nested layer mapping to a specific focus level. For ADHD hyperfocus, hierarchical nesting allows users to collapse all layers except the target, eliminating visual and cognitive distraction. Hyperfocus locks onto a single nested layer, with the structure providing clear boundaries to prevent layer-switching.
