#+CATEGORY: Concept Info Organization Types > Abstraction Layered

* Definition
Abstraction-layered organization uses DIKW-like layers of increasing abstraction, structuring information from raw data (lowest abstraction) to actionable wisdom (highest abstraction). Each layer adds semantic richness to the layer below.

* ADHD/Hyperfocus/Russian Doll Compatibility
Abstraction layers mirror Russian Doll nesting, with each layer encapsulating the lower abstraction layer. For ADHD hyperfocus, users can lock onto a single abstraction layer (e.g., Knowledge) hiding higher and lower layers to prevent distraction. The clear layer boundaries align with hyperfocus's preference for bounded, well-defined work scopes.
