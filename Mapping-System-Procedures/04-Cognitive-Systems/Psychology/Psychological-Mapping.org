#+TITLE: Psychological Mapping Methods
#+CATEGORY: Psychology > Psychological Mapping Methods

* Definition
Psychological mapping methods visualize mental processes, brain-behavior relationships, and psychological constructs using techniques like connectograms, cognitive maps, and psychometric profiling.

* ADHD/Hyperfocus/Russian Doll Compatibility
Maps cognitive processes underlying ADHD hyperfocus (attention networks, executive function). Russian doll layers: behavior → cognitive process → brain network → psychological model.
