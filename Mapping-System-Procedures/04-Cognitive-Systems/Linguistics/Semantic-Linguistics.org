#+TITLE: Semantic Linguistics
#+CATEGORY: Linguistics > Semantic Linguistics

* Definition
Semantic linguistics studies meaning in language, including semantic networks, frames, and conceptual spaces, often using computational models to map semantic relationships.

* ADHD/Hyperfocus/Russian Doll Compatibility
Semantic networks/concept maps align with concept thinking. Russian doll layers: words → semantic features → networks → semantic fields.
