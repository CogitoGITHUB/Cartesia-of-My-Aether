#+TITLE: Cognitive Linguistics
#+CATEGORY: Linguistics > Cognitive Linguistics

* Definition
Cognitive linguistics studies language as embedded in cognition, examining how conceptual structures, metaphor, image schemas, and construction grammar shape meaning and communication.

* ADHD/Hyperfocus/Russian Doll Compatibility
Concept-centric language analysis aligns with concept thinking. Russian doll layers: lexical items → constructions → conceptual domains → cognitive models. Helps hyperfocus on linguistic concepts.
