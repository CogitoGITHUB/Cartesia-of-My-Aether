#+TITLE: Linguistic Mapping Methods
#+CATEGORY: Linguistics > Linguistic Mapping Methods

* Definition
Linguistic mapping methods visualize cross-linguistic semantic patterns using semantic maps, conceptual spaces, and typological databases to compare meaning structures.

* ADHD/Hyperfocus/Russian Doll Compatibility
Maps linguistic concepts hierarchically/networked, aligning with concept thinking. Russian doll layers: lexical items → semantic maps → typological patterns → language families.
