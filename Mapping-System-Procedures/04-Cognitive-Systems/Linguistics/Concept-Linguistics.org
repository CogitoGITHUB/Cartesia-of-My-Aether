#+TITLE: Concept Linguistics
#+CATEGORY: Linguistics > Concept Linguistics

* Definition
Concept linguistics focuses on how linguistic structures encode and organize conceptual categories, using semantic maps, frames, and conceptual spaces to model meaning cross-linguistically.

* ADHD/Hyperfocus/Russian Doll Compatibility
Directly aligns with concept thinking: maps linguistic concepts hierarchically. Russian doll layers: lexical units → concepts → semantic maps → linguistic typology.
