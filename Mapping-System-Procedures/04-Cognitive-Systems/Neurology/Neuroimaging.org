#+TITLE: Neuroimaging Techniques
#+CATEGORY: Neurology > Neuroimaging Techniques

* Definition
Neuroimaging techniques are tools to visualize brain structure (sMRI, CT) and function (fMRI, PET, EEG, MEG) for research and clinical applications, including ADHD diagnosis.

* ADHD/Hyperfocus/Russian Doll Compatibility
Key for ADHD diagnosis and hyperfocus research: fMRI shows default mode network alterations during hyperfocus. Russian doll layers: scan → region → network → diagnosis.
