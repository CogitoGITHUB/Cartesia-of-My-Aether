#+TITLE: Brain Mapping Methods
#+CATEGORY: Neurology > Brain Mapping Methods

* Definition
Brain mapping methods localize and visualize brain structure, function, and connectivity using techniques like fMRI, EEG, MEG, PET, and diffusion MRI to create detailed brain atlases.

* ADHD/Hyperfocus/Russian Doll Compatibility
Direct application to ADHD: maps hyperfocus-related networks (default mode network, frontal control network). Russian doll layers: voxel → region → network → system maps.
