* Rules Overview

The Rules define operating constraints and principles for the Mapping-System-Procedures system.

* Core Rule Hierarchy

| Level | Description | Override Authority |
|-------|-------------|------------------|
| Core Rules | Foundation constraints | None (absolute) |
| Layer Rules | Domain constraints | Architect |
| Local Rules | Specific situation rules | Self within domain |

* Three Fundamental Rules

1. **Return to The Self** - When uncertain, return to center
2. **The Architect Acts** - The Self observes; the Architect acts
3. **Layer Clarity** - Maintain explicit layer boundaries
