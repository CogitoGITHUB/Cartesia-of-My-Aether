#+TITLE: Cognitive Task Analysis
#+CATEGORY: Scientific Methods & Info Mapping > Cognitive Mapping

* Definition
Cognitive Task Analysis (CTA) examines expert mental models and task workflows to understand cognitive processes underlying complex task performance. It focuses on uncovering implicit knowledge, decision-making strategies, and perceptual cues used by experts, distinguishing it from traditional task analysis which focuses on observable behaviors.

* ADHD/Hyperfocus/Russian Doll Compatibility
CTA aligns with Russian Doll structures by decomposing expert workflows into nested, hierarchical layers of cognitive complexity. For ADHD hyperfocus, CTA provides structured task breakdowns that reduce cognitive load, allowing hyperfocus to engage on specific layers without distraction. The nested CTA outputs mirror Russian Doll nesting, where each layer builds on the previous, enabling hyperfocus to lock onto a single layer while maintaining awareness of the broader structure.
