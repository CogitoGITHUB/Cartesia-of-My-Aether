#+TITLE: DIKW Pyramid
#+CATEGORY: Information Mapping Methods > Layered Abstraction Methods

* Definition
The DIKW (Data-Information-Knowledge-Wisdom) pyramid is a hierarchical model where each level builds upon the previous: data → information → knowledge → wisdom, forming nested layers of abstraction.

* ADHD/Hyperfocus/Russian Doll Compatibility
Perfect Russian doll layering: each layer is a doll nested inside the previous. Concept-centric abstraction aligns with concept thinking. Hyperfocus can be applied to a single layer's refinement.
