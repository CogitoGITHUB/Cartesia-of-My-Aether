#+TITLE: Gantt Charts & Kanban
#+CATEGORY: Information Mapping Methods > ADHD-Optimized Methods

* Definition
Gantt charts are bar charts illustrating project schedules; Kanban is a visual workflow method using columns to represent task stages. Both are task-oriented project management tools.

* ADHD/Hyperfocus/Russian Doll Compatibility
Poor match for Russian doll layering: they are flat, task-based, not concept-centric. Not recommended for ADHD hyperfocus as they encourage context switching and fragment attention.
