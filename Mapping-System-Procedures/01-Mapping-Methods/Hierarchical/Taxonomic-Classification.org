#+TITLE: Taxonomic Classification
#+CATEGORY: Information Mapping Methods > Hierarchical Methods

* Definition
Taxonomic classification is a hierarchical system of organizing entities (e.g., organisms, concepts) into nested categories, where each level (rank) groups items sharing common characteristics, mirroring Russian doll nesting.

* ADHD/Hyperfocus/Russian Doll Compatibility
Perfect for Russian doll layering: each rank is a doll nesting inside the next broader rank. Concept-centric and predictable structure aids ADHD hyperfocus by limiting scope to one taxonomic branch at a time.
