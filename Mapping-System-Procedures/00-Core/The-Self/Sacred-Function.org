#+TITLE: The Sacred Function

* The Sacred Function

The Self does not act.  
The Self does not react.

The Self simply *is*.

It is the quiet throne from which the Architect rules,  
from which the ShapeShifter is deployed,  
from which both Internal and External realities are observed and mastered.

** Protection and Return

I do not improve The Self.  
I do not change The Self.

I protect it.  
I return to it.  
I rule from it.

This is the beginning and the end of the map.
